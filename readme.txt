== Description ==

Increasing website performance and reducing load time.

Very easy to install and very easy to use.

You don't need to be a developer to understand and configure this plugin. 

Website Performance Optimization technology for different CMS like WordPress, Joomla and etc.
Trusted by millions of publishers, web developers, and web hosts worldwide. It is the performance solution for optimizing improvements in search engine result page rankings.
At least 10x improvement in overall site performance. Reduced page load time: increased visitor time on site. Visitors view more pages. Improves web server performance.

This plugin generates static html files from your dynamic WordPress blog. After a html file is generated your webserver will serve that file instead of processing the comparatively heavier and more expensive WordPress PHP scripts.

99% of your visitors will be served static html files. One cached file can be served thousands of times. Other visitors will be served custom cached files tailored to their visit. If they are logged in, or have left comments those details will be displayed and cached for them.

Speedup supports WordPress Multisite and is compatible with most popular plugins, including WooCommerce, bbPress, Yoast SEO and other popular plugins.

General Features may be used by anyone with any web server (LiteSpeed, Apache, NGINX, etc.)
